# OKR Portál - Interný Zamestnanecký Systém

Moderná webová aplikácia pre správu zamestnancov, rozpisov pohotovostí, cestovných príkazov a príspevkov úrazového poistenia.

## 📋 Popis

OKR Portál je integrovaný AI systém určený pre interné procesy organizácie, ktorý kombinuje správu ľudských zdrojov s plánovacími a administratívnymi funkciami. Aplikácia využíva Firebase Firestore ako cloudovú databázu a poskytuje moderné používateľské rozhranie s tmavým dizajnom.

Aplikácia je vyvíjaná interným tímom OKR Banská Bystrica.
